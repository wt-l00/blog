# runc

# はじめに

runcやrunscなどのruntimeに興味があるので、記事として書く。 なにはともあれ、まずは動かしてみる。
opencontainer runc (opens new window)に従います。

# building

ローカルを汚したくないので、Vagarntの環境で動かします。

Vagrantfileは以下。 ubuntu18.04を使います。

Vagrant.configure("2") do |config|
  config.vm.box = "generic/ubuntu1804"
  config.vm.provision :shell, :path => "./install.sh"
end

#! /bin/sh
sudo apt update
# install dependency
sudo apt -y install gcc pkg-config libseccomp-dev
# install golang
wget https://dl.google.com/go/go1.14.2.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.14.2.linux-amd64.tar.gz
rm go1.14.2.linux-amd64.tar.gz
echo "export PATH=$PATH:/usr/local/go/bin" >> .bashrc
echo "export GOPATH=/home/vagrant/go" >> .bashrc
## install docker
sudo apt -y install docker.io
# add docker user group
sudo usermod -aG docker $USER
sudo systemctl start docker
sudo systemctl enable docker
